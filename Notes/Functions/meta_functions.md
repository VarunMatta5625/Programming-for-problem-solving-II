# Meta Functions

## type()
type function returns the data type of the variable <br>
For example,  <br>

 ```
a = 65
b = int(a)
c = chr (a)

print(type(a))
print(type(b))
print(type(c))
```
Outputs to, <br>
`<class 'int'>`<br>
`<class 'int'>`<br>
`<class 'str'>`<br>

<br>

`type()` can also be used to identify lists
```
a = []
print(type(a))
```

Outputs to, <br>
`<class 'list'>`
